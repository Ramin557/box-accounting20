    کنترل‌های حالت تیره

    سیستم حالت تیره پیشرفته با انتقال‌های نرم و ذخیره خودکار تنظیمات کاربر

    روش‌های تغییر حالت:
    کلیک روی دکمه سوئیچ (بالای صفحه)
    میانبر کیبورد Ctrl + Shift + D
    تشخیص خودکار سیستم
    ذخیره خودکار تنظیمات
    ویژگی‌های پیشرفته:
    • انتقال نرم بین حالت‌ها (300ms)
    • حفظ تنظیمات در LocalStorage
    • تشخیص خودکار تم سیستم
    • پشتیبانی از تمام اجزای UI
    • نوتیفیکیشن زیبای تغییر حالت
    • سازگاری با اسکرین ریدر
    نکته: برای تست بهتر، حالت تیره را فعال کنید و تمام بخش‌های سیستم را بررسی کنید. تنظیمات شما به صورت خودکار ذخیره می‌شود.

    اطلاعات جدید در دسترس است.
    جزئیات فنی پیاده‌سازی
    کلاس‌های CSS:
    • .dark-mode - کلاس اصلی حالت تیره
    • .theme-transitioning - انیمیشن انتقال
    • .dark-mode-toggle - دکمه سوئیچ
    • .dark-mode-notification - نوتیفیکیشن
    API عمومی:
    • darkModeManager.toggle() - تغییر حالت
    • darkModeManager.setDarkMode(true/false) - تنظیم مستقیم
    • darkModeManager.getCurrentTheme() - دریافت حالت فعلی
    • window.addEventListener('themeChanged') - رویداد تغییر
    نحوه استفاده در کد:
    // تغییر حالت به صورت برنامه‌نویسی
window.darkModeManager.setDarkMode(true);

// گوش دادن به تغییرات تم
window.addEventListener('themeChanged', (e) => {
    console.log('New theme:', e.detail.isDarkMode ? 'dark' : 'light');
});

// دریافت وضعیت فعلی
const currentTheme = window.darkModeManager.getCurrentTheme();
console.log('Current theme:', currentTheme);
